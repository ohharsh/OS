Page Replacement Algorithms decide which memory page is to be replaced. The process of replacement is sometimes called swap out or write to disk. Page replacement is done when the requested page is not found in the main memory (page fault).

FIFO

First In, First Out page replacement algorithm

LRU

Least Recently Used page replacement algorithm

Optimal

A theoretical page replacement algorithm
